# Agent instructions (Cursor)

This repository is configured for **Cursor**. All AI-assisted development must follow **[docs/base-standards.mdc](docs/base-standards.mdc)** as the single source of truth for rules, standards, and skills discovery.

- **Project rules:** `.cursor/rules/` (for example `use-base-rules.mdc`)
- **Commands:** `.cursor/commands/` (content in `specs/.commands/`)
- **Agents:** `.cursor/agents/` (definitions in `specs/.agents/`)
- **Skills:** `specs/skills/`
